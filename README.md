## 🚀 Features

| Feature | Description |
|---------|-------------|
| ⛏️ **Real-time Mining** | Watch mining operations live |
| 👷 **5 Active Miners** | Workers mining 24/7 automatically |
| 📊 **Live Statistics** | Shares, Blocks, Hashrate updated every second |
| 📈 **Hashrate Chart** | 60-second animated history graph |
| 🎉 **Blocks Tracker** | See when blocks are found |
| 📱 **Responsive Design** | Works on all devices |

## 📊 Dashboard Stats

- **Active Miners** - Number of workers
- **Total Shares** - Shares submitted
- **Blocks Found** - Blocks discovered
- **Pool Hashrate** - GH/s rate
- **Mining Uptime** - Time running
- **Difficulty** - Current difficulty

## 🔧 Local Development

```bash
# Clone the repository
git clone https://github.com/yayass3r/bitcoin-mining-dashboard.git

# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```
